# Gitfolio AI 🚀

Turn your GitHub into a job-ready developer profile in seconds.

## ✨ Overview

Gitfolio AI is an AI-powered GitHub portfolio analyzer that helps developers present their work better.  
It analyzes your repositories and generates:

- 🧠 Developer summary
- 💪 Strengths & skills
- 🛠 Tech stack breakdown
- 📄 Resume-ready content

Perfect for students and developers preparing for internships and job applications.

---

## 🔥 Features (MVP)

- GitHub profile analysis
- AI-generated developer summary
- Strengths & skill insights
- Clean, shareable output

---

## 🛠 Tech Stack

### Frontend
- Next.js
- Tailwind CSS

### Backend
- FastAPI (Python)
- PostgreSQL

### AI
- OpenAI API (or compatible LLM)

---

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/your-username/gitfolio-ai.git
cd gitfolio-ai
```

### 2. Setup backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Setup frontend
```bash
cd frontend
npm install
npm run dev
```
